# Hi, I'm Tom Sander 👋

**Ph.D. Student in Computer Vision & Machine Learning** at TU Dortmund University, focusing on **multimodal learning in planetary science**. I build deep learning systems that make sense of remote sensing data from the Moon and beyond.

## 🔭 What I work on

- Deep learning for planetary remote sensing, including detecting Apollo landing sites and analyzing lunar swirls from multimodal satellite data
- Multimodal learning that combines heterogeneous data sources into unified models
- Classical and modern optimization, from saliency algorithms to custom PyTorch implementations

## 📌 Featured projects

| Project | Description |
|---|---|
| [lunar-technosignatures](https://github.com/TechnicToms/lunar-technosignatures) | Deep learning approach to detecting Apollo landing sites on the lunar surface, with an accompanying research paper |
| [multimodal-lunar-swirls](https://github.com/TechnicToms/multimodal-lunar-swirls) | Investigating lunar swirls by modeling relationships between different remote sensing modalities |
| [simple-levenberg-marquardt-torch](https://github.com/TechnicToms/simple-levenberg-marquardt-torch) | A clean PyTorch implementation of the Levenberg-Marquardt optimization algorithm |
| [saliency](https://github.com/TechnicToms/saliency) | A collection of saliency map algorithms, including Itti's method |
